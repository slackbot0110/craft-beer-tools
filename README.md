## 包含工具

### ✍️ 精酿笔记生成器

输入酒款关键信息，一键生成小红书爆款文案。

- 🎯 **智能标题**：根据评价倾向（好评/中评/差评）自动选择标题策略
- ✨ **感官美化**：40+ 闻香关键词、30+ 口感关键词自动改写为画面感表达
- 💬 **人话翻译**：专业描述自动附带大白话版本，降低阅读门槛
- 📋 **一键复制**：标题 / 正文 / 标签分别复制或全部复制，直接粘贴到小红书
- 📦 **四种类型**：单酒测评、合集推荐、趋势热点、探店

### 📅 30天内容排期表

精心规划的 30 天内容日历，按「7:2:1」内容矩阵分配。

- 📊 四周递进策略（建立节奏 → 测试类型 → 复制成功 → 放大爆款）
- 🕐 每天的最佳发布时间建议
- 🏷️ 每篇的标签推荐
- 📈 数据复盘检查点
